RESTAURANT MANAGEMENT SYSTEM - PROJECT SUMMARY FOR EXTERNAL AI

## Project Overview
A comprehensive restaurant management dashboard for Smash Bros Burgers (Rawai, Thailand) with AI-powered analytics and Loyverse POS integration.

## Key Technologies
- Frontend: React 18 + TypeScript + Vite
- Backend: Node.js + Express + TypeScript
- Database: PostgreSQL + Drizzle ORM
- UI: Tailwind CSS + shadcn/ui components
- AI: OpenAI GPT-4o integration
- POS: Loyverse API integration
- Email: Gmail API
- Routing: Wouter

## Architecture
```
client/
├── src/
│   ├── components/ui/          # shadcn/ui components
│   ├── components/             # Custom components
│   ├── pages/                  # Page components
│   ├── lib/                    # Utilities
│   └── hooks/                  # Custom hooks

server/
├── services/                   # Business logic
├── routes/                     # API routes
├── constants/                  # Configuration
└── storage.ts                  # Database interface

shared/
└── schema.ts                   # TypeScript schemas
```

## Key Features
1. **Dashboard**: Real-time KPIs, sales analytics, AI insights
2. **Daily Stock & Sales**: Staff reporting forms with email notifications
3. **Shopping List**: Automated procurement management
4. **Recipe Management**: Menu items with ingredient tracking
5. **Loyverse Integration**: Live POS data sync
6. **AI Analytics**: Receipt analysis and anomaly detection
7. **Expense Tracking**: Business expense management
8. **Shift Analytics**: 5pm-3am shift cycle tracking (Bangkok timezone)
